## 核心功能

### 1. 地图展示
- SVG 矢量地图，清晰美观
- 支持缩放（1x-3x）
- 支持平移（放大后）
- 边界约束，防止移出可视区域

### 2. 建筑标记与详情
- 所有主要建筑均有标记
- 点击建筑显示详情弹窗
- 弹窗包含建筑图片和详细说明

### 3. 路线导航
- 点击任意建筑显示路线
- 小女孩沿路线行走动画（8秒）
- 动画流畅，无冲突
- 支持刷新清除路线

### 4. 背景音乐
- 校园欢迎歌
- 国歌（升旗台点击播放）
- 播放/暂停控制

### 5. 响应式设计
- 支持桌面端和移动端
- 触控优化
- 自适应布局

### 6. 视觉效果
- 3D童话风格建筑
- 鲜艳的色彩
- 学校东侧小河景观
- 光影效果

## 使用说明

### 地图操作
- **缩放**: 使用鼠标滚轮或双指捏合
- **平移**: 放大后拖动地图
- **查看建筑**: 点击建筑图标
- **路线导航**: 点击建筑后自动显示路线和小女孩动画
- **播放音乐**: 点击右上角音乐按钮或升旗台

### 移动端操作
- **缩放**: 双指捏合
- **平移**: 单指拖动（放大后）
- **扫码分享**: 点击页脚"📱 扫码分享"查看二维码
